# Ly Hong Phuong An - Portfolio Website

A modern, responsive portfolio website built with React, TypeScript, TailwindCSS, and Vite.

## 🚀 Live Demo

**[https://lyhongphuongan.github.io](https://lyhongphuongan.github.io)**

## ✨ Features

- ⚡ **Fast** - Built with Vite for lightning-fast development and builds
- 🎨 **Modern UI** - Clean design with animations using TailwindCSS
- 🌓 **Dark/Light Mode** - Toggle between themes
- 📱 **Responsive** - Works on all devices
- 🔍 **SEO Optimized** - Meta tags, Open Graph, Schema.org
- 📲 **PWA Ready** - Installable as an app
- 🖼️ **Lazy Loading** - Optimized image loading

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript
- **Styling**: TailwindCSS, shadcn/ui
- **Build Tool**: Vite
- **Deployment**: GitHub Pages

## 📦 Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## 📁 Project Structure

```
├── public/          # Static assets
│   ├── images/      # Images and icons
│   └── cv/          # Resume/CV files
├── src/
│   ├── components/  # React components
│   │   ├── portfolio/  # Portfolio sections
│   │   └── ui/         # UI components (shadcn)
│   ├── data/        # Data files (config, projects, skills)
│   ├── pages/       # Page components
│   └── lib/         # Utilities
└── index.html       # Entry point
```
